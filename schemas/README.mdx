# JSON Schemas

All schemas define the message formats used in the Agentic Intent Protocol (AIP).
Each schema is versioned independently and validated through conformance tests.

| Schema | Description |
|--------|--------------|
| [Context Request](context-request.mdx) | User intent data from platform |
| [Bid](bid.mdx) | Advertiser agent bid payload |
| [Auction Result](auction-result.mdx) | Winning bid and serve token |
| [Event - CPX](event-cpx.mdx) | Verified exposure |
| [Event - CPC](event-cpc.mdx) | Verified click |
| [Event - CPA](event-cpa.mdx) | Verified conversion |
| [Ledger Record](ledger-record.mdx) | Monetary ledger entry |
| [Wallet Record](wallet-record.mdx) | Account balance and transaction |
| [Version](version.mdx) | Protocol version metadata |
