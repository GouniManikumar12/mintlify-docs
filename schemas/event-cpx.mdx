# Event - CPX Schema

## Overview
Captures a verified exposure event generated by an AI platform.

## Schema Fields
| Field | Type | Description |
|--------|------|-------------|
| serve_token | string | Unique ID linking exposure to auction |
| event_type | string | Must equal "exposure" |
| timestamp | string | RFC3339 timestamp |
| signature | string | Cryptographic verification |
| metadata | object | Context like platform_id, agent_id |

## Example
```json
{
  "serve_token": "srv_20381f",
  "event_type": "exposure",
  "timestamp": "2025-11-12T01:41:00Z",
  "signature": "a3f0ab92",
  "metadata": {"platform_id": "plat_ai_example", "agent_id": "brand_quickbooks"}
}
```
