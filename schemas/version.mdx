# Version Schema

## Overview
Provides versioning metadata for AIP implementations.

## Schema Fields
| Field | Type | Description |
|--------|------|-------------|
| current | string | Active version |
| supported | array | List of supported versions |
| deprecated_after | string | Deprecation cutoff date |

## Example
```json
{
  "current": "1.0",
  "supported": ["1.0"],
  "deprecated_after": "2026-06-30"
}
```
