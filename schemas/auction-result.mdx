# Auction Result Schema

## Overview
Defines the result of a completed AIP auction with the winning Brand Agent.

## Schema Fields
| Field | Type | Description |
|--------|------|-------------|
| auction_id | string | Unique auction identifier |
| serve_token | string | Token linking all future events |
| winner | object | Winning bid details |
| timestamp | string | RFC3339 timestamp |

## Example
```json
{
  "auction_id": "auc_92b134",
  "serve_token": "srv_20381f",
  "winner": {
    "agent_id": "brand_quickbooks",
    "score": 0.87,
    "price": 0.05,
    "creative": "Try QuickBooks for Startups",
    "landing_url": "https://quickbooks.example"
  },
  "timestamp": "2025-11-12T01:40:00Z"
}
```
