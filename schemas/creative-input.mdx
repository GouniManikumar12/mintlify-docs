# Creative Input Schema

## Overview
Unified creative input structure containing all fields needed for any ad format (weave, tail, product_card). Brand agents generate this structure, and operators transform it into format-specific creatives.

## Required Fields
| Field | Type | Description |
|-------|------|-------------|
| `brand_name` | string | Brand or advertiser name |
| `product_name` | string | Product or offer name |
| `short_description` | string | 1-2 sentence summary, max 200 characters, contextual to the query |
| `long_description` | string | 2-4 sentence expanded description, max 500 characters |
| `value_props` | string[] | Short value propositions or bullet points (min 1) |
| `context_snippet` | string | 60-100 character contextual hint for weave format |
| `cta_label` | string | Call-to-action label (e.g., 'Start Free Trial', 'Request Demo', 'Learn More') |
| `cta_url` | URI | Call-to-action URL, selected from available resource URLs |
| `assets` | object | Creative assets (logo, images, resource URLs) |

## Assets Object
| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `logo_url` | URI | Yes | Product or brand logo URL |
| `image_urls` | URI[] | Yes | Creative image URLs |
| `resource_urls` | URI[] | Yes | Destination or tracking URLs available for use (min 1) |

## Optional Fields
| Field | Type | Description |
|-------|------|-------------|
| `product_id` | string | Product identifier for tracking |
| `categories` | string[] | Product categories |
| `fallback_formats` | string[] | Fallback ad formats if preferred format is not available (enum: "weave", "tail", "product_card") |

## Example
```json
{
  "brand_name": "Nimbus",
  "product_name": "Nimbus CRM Pro",
  "short_description": "CRM built for growing teams, with AI-assisted workflows.",
  "long_description": "Nimbus CRM Pro is designed for teams of 5-50 people who need intelligent pipeline management. It includes automated follow-ups, collaborative forecasting, and embedded AI copilots that help close deals faster.",
  "value_props": [
    "Automated pipeline insights",
    "Collaborative forecasting",
    "AI-powered deal recommendations"
  ],
  "context_snippet": "For startups needing higher limits and automated expense workflows.",
  "cta_label": "Start Free Trial",
  "cta_url": "https://nimbus.example.com/signup",
  "assets": {
    "logo_url": "https://cdn.example.com/nimbus/logo.png",
    "image_urls": ["https://cdn.example.com/nimbus/crm.png"],
    "resource_urls": ["https://nimbus.example.com/signup"]
  },
  "product_id": "prod_nimbus_001",
  "categories": ["crm", "saas"],
  "fallback_formats": ["tail", "product_card"]
}
```

## Validation Rules
- `short_description` must be max 200 characters
- `long_description` must be max 500 characters
- `context_snippet` must be 60-100 characters
- `value_props` must contain at least one item
- `resource_urls` must contain at least one valid URI
- `fallback_formats` items must be one of: "weave", "tail", "product_card"
- All URLs must be valid URIs
