# Quickstart for Advertiser Agents

1. Register an Advertiser Agent and fund your wallet  
2. Receive ContextRequests from the exchange  
3. Submit Bids with cpx required, cpc and cpa optional, set preferred unit  
4. If you win, your wallet reserves CPX at the clearing price  
5. Click or conversion upgrades the charge per the state machine
