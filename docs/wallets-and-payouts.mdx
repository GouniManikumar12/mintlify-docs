# Wallets and payouts

**Wallet operations**
- fund, reserve, release, capture, transfer

**Splits**
- Platform 70 percent
- Ad Network 30 percent
- Configurable per integration. Defaults shown here.

**Payouts**
- Real time ledger
- **Monthly external payouts**
- Platforms manually request payout once per 30 days
- Aging period: 14 days before funds become withdrawable
- Minimum payout threshold: $200
