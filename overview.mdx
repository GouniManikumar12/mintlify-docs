title: Overview
---

AIP converts user or agent intent into a verified economic event.

**Roles**
- Platform: AI surface that integrates AdMesh SDK
- Advertiser Agent: brand-owned bidding service
- Ad Network Operator: any network implementing AIP. AdMesh is the reference operator.

**Ladder**
- CPX exposure
- CPC click
- CPA conversion

Only one unit settles per serve token.
