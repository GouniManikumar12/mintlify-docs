## Platform Integration

### Step 1: Understand the Flow

As a platform, you'll:
1. Send context requests when users ask questions
2. Receive ad recommendations from networks
3. Display ads to users
4. Track events (exposure, clicks, conversions)

### Step 2: Install Dependencies

```bash
npm install ajv  # For JSON schema validation
```

### Step 3: Send Context Requests

```javascript
// When user asks a question
const contextRequest = {
  session_id: generateUUID(),
  message_id: generateUUID(),
  user_query: "best running shoes for marathon training",
  platform_id: "your-platform-id",
  timestamp: new Date().toISOString()
};

// Send to ad network
const response = await fetch('https://network.example.com/api/context', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': `Bearer ${API_KEY}`
  },
  body: JSON.stringify(contextRequest)
});

const recommendations = await response.json();
```

### Step 4: Display Recommendations

```javascript
// Display the winning ad
if (recommendations.winner) {
  displayAd({
    title: recommendations.winner.ad_content.title,
    description: recommendations.winner.ad_content.description,
    url: recommendations.winner.ad_content.landing_url
  });
}
```

### Step 5: Track Events

```javascript
// Track exposure
await trackEvent({
  event_type: 'exposure',
  session_id: contextRequest.session_id,
  message_id: contextRequest.message_id,
  recommendation_id: recommendations.winner.bid_id,
  timestamp: new Date().toISOString()
});

// Track click (when user clicks)
await trackEvent({
  event_type: 'click',
  session_id: contextRequest.session_id,
  message_id: contextRequest.message_id,
  recommendation_id: recommendations.winner.bid_id,
  timestamp: new Date().toISOString()
});
```

## Network Integration

### Step 1: Set Up API Endpoints

Create these endpoints:

- `POST /api/context` - Receive context requests
- `POST /api/events` - Receive tracking events
- `GET /api/health` - Health check

### Step 2: Implement Auction Logic

```javascript
async function runAuction(contextRequest) {
  // 1. Request bids from advertisers
  const bids = await requestBids(contextRequest);
  
  // 2. Validate bids
  const validBids = bids.filter(bid => validateBid(bid));
  
  // 3. Run second-price auction
  const sortedBids = validBids.sort((a, b) => b.amount - a.amount);
  const winner = sortedBids[0];
  const secondPrice = sortedBids[1]?.amount || winner.amount;
  
  // 4. Return result
  return {
    auction_id: generateUUID(),
    winner_bid_id: winner.bid_id,
    winning_amount: secondPrice,
    timestamp: new Date().toISOString()
  };
}
```

### Step 3: Handle Events

```javascript
app.post('/api/events', async (req, res) => {
  const event = req.body;
  
  // Validate event
  if (!validateEvent(event)) {
    return res.status(400).json({ error: 'Invalid event' });
  }
  
  // Store event
  await storeEvent(event);
  
  // Update ledger
  if (event.event_type === 'click') {
    await updateLedger(event);
  }
  
  res.json({ success: true });
});
```

### Step 4: Implement Settlement

```javascript
async function settleDailyPayouts() {
  const events = await getEventsForSettlement();
  
  for (const event of events) {
    const ledgerRecord = {
      record_id: generateUUID(),
      transaction_type: 'ad_revenue',
      amount: event.winning_amount,
      currency: 'USD',
      timestamp: new Date().toISOString(),
      related_event_id: event.event_id,
      settlement_status: 'pending'
    };
    
    await createLedgerRecord(ledgerRecord);
  }
}
```

## Advertiser Integration

### Step 1: Register with Network

Contact an AMXP-compatible network to get API credentials.

### Step 2: Receive Bid Requests

```javascript
app.post('/api/bid-request', async (req, res) => {
  const contextRequest = req.body;
  
  // Decide if you want to bid
  if (!shouldBid(contextRequest)) {
    return res.status(204).send();
  }
  
  // Create bid
  const bid = {
    bid_id: generateUUID(),
    advertiser_id: 'your-advertiser-id',
    context_request_id: contextRequest.message_id,
    amount: calculateBidAmount(contextRequest),
    currency: 'USD',
    ad_content: {
      title: "Your Product Name",
      description: "Compelling description",
      landing_url: "https://yoursite.com/product",
      image_url: "https://yoursite.com/image.jpg"
    },
    timestamp: new Date().toISOString()
  };
  
  res.json(bid);
});
```

### Step 3: Track Performance

```javascript
// Monitor your ad performance
const performance = await fetch('https://network.example.com/api/advertiser/stats', {
  headers: {
    'Authorization': `Bearer ${API_KEY}`
  }
});

const stats = await performance.json();
console.log('Impressions:', stats.exposures);
console.log('Clicks:', stats.clicks);
console.log('Conversions:', stats.conversions);
console.log('Spend:', stats.total_spend);
```

## Validation

### Validate Payloads

Use JSON schemas to validate all payloads:

```javascript
import Ajv from 'ajv';
import contextRequestSchema from './schemas/context-request.json';

const ajv = new Ajv();
const validate = ajv.compile(contextRequestSchema);

const isValid = validate(payload);
if (!isValid) {
  console.error('Validation errors:', validate.errors);
}
```
