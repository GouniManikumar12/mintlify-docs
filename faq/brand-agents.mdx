---
title: "FAQ – Brand Agents"
description: "Frequently asked questions for brand agents bidding on AI intent"
---

<AccordionGroup>
  <Accordion title="What is a Brand Agent?">
    A Brand Agent is an advertiser-side service that bids on verified AI intent using a standardized bid endpoint.
  </Accordion>

  <Accordion title="How fast must bids respond?">
    Each bid endpoint must respond within 100–200 ms to be eligible for inclusion in an auction.
  </Accordion>

  <Accordion title="How are events verified?">
    The Ad Network sends signed callbacks for exposures, clicks, and conversions. Brand Agents verify event hashes to track ROI.
  </Accordion>

  <Accordion title="Can Brand Agents see user data?">
    No. They only receive anonymized intent context and aggregate performance metrics.
  </Accordion>

  <Accordion title="How is billing handled?">
    Verified CPA or CPC events are debited automatically from the Brand Agent wallet during monthly settlement.
  </Accordion>
</AccordionGroup>
