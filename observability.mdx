---
title: Observability
---

**Metrics**
- Auction p95 ≤ 20 ms
- Event ingest p95 ≤ 50 ms
- Duplicate event rate < 0.01 percent
- Wallet insufficient rate < 1 percent

**Logs**
- Structured, immutable, serve_token keyed

**Alerts**
- Latency SLO breaches
- Signature failures
- Clicks without exposures
